### graph-units → цепочка (ДНК / РНК) → клетка → орган → жизнь

## 1. Что это

Универсальный метод: любой объект — гипотеза, функция, процесс, план, схема — превращается в граф ветвления и работающую на нём динамику. Один способ на все классы; различия классов вынесены в типизацию отношений. «Оцифровка» здесь — не перевод в числа, а извлечение скрытого графа и его работы.

## 2. Главный принцип: сворачивание, не сборка

Метод идёт **сверху вниз**. Структуру не собирают из кубиков — её сворачивают до неразложимого ядра:

```
объект → проекция орграфа в граф → снятие симметрии → снятие фрактала → ЯДРО
```

Ядро — точка, где редукция встала. Дальше не сжимается.

## 3. graph-unit (ядро)

Ядро (graph-unit) — базовая единица построения, аналог основания ДНК (в биологии оснований 4).

**Определение через отрицание:** асимметричная пространственная фигура без симметрии и без фрактальности на уровень ниже. Логика точна — симметрия и самоподобие суть сжимаемость; выжав всё сжимаемое, получаешь несжимаемый остаток. Это алгоритмическая логика: структура = избыточность, ядро = чистая информация; несжимаемое по природе выглядит асимметричным.

Ядер **мало** — алфавит конечен. Гипотеза: их число зависит от пространственной размерности фигуры — **2, 3, 4** (4 = глубина, четвёртая ось для 3D; математически ℝ⁴ законно, хоть и невизуализируемо). Библиотека ядер пополняется в процессе работы по протоколу.

*Статус:* сам graph-unit пока гипотеза — задан как процесс вычисления, проявляющийся при редукции, а не как заранее известный символ.

## 4. Три оси декора

Ядро — только форма. Реальный объект надевает на форму три отдельные оси при сборке:

> **объект = цепочка ядер + направления + параллели + строб(время)**

- **направления** — причинность (ориентация рёбер);
- **параллели** — кратность связей;
- **строб(время)** — стробоскоп: замораживает динамику в последовательность статических кадров.

Редукция снимала и время, и направление именно потому, что снимала *всю* декорацию, докапываясь до формы. Единый принцип: **алфавит = формы, реальность = формы + три оси декора.**

## 5. Цепочка и ветвление (генотип / фенотип)

Цепочка ядер (1D) — **код (генотип)**. Ветвление — то, что код разворачивает (**фенотип**). Как в биологии: 1D-последовательность оснований → многомерная свёрнутая структура.

Здесь источник сжатия и экономии FLOPs: крошечный алфавит + длинные цепочки = безграничные структуры. Свойства считаются один раз на ядро и переиспользуются. Природа это уже доказала — 4 основания порождают всё живое.

## 6. ДНК и РНК

Одни и те же ядра строят оба носителя:

- **ДНК = структура** — фигура (статична, dim 2/3/4);
- **РНК = работа** — стробо-последовательность фигур.

Это разрешает исходный тезис «динамика может быть статична»: каждый кадр неподвижен, движение = последовательность кадров.

## 7. Уровни: ДНК → клетка → орган → жизнь

Уровни **по-настоящему независимы** — не матрёшка. Орган собирается напрямую из ядер; знать строение клетки не обязательно (прямое действие). Клетка и орган — разные типы в одной библиотеке, а не вложенные слои. Принцип: **простота = здоровье, сложность = болезнь.**

- **Клетка** — операционально замкнута: петля ДНК → работа → ДНК замкнута на собственный код (закрывает регресс «кто чинит ремонтёра»). Отличает живое от мёртвой формы.
- **Орган** — более крупная самодостаточная единица, собранная *своим* кодом, а не склейкой клеток.
- **Жизнь / поле** — среда взаимодействия и отбора моделей.

## 8. Слияние: 1 + 1 = 1

Объединение графов — всегда **новая цельность**, не сумма (1+1=1; 1+1+1+2=1). Слияние = рекомбинация цепочек: две цепочки дают новую (новый набор ДНК), не склейку встык. Библиотека **генеративна** — новые ядра всплывают по ходу работы, но остаются немногими.

## 9. Избирательность

Оцифровать всё нельзя: комбинаторика бьёт FLOPs по степенному закону. Оцифровывается только интересное исследователю — под запрос строится механизм познания. Поле знаний **ленивое**: модель существует, только если её запросили. (Неконтролируемый рост = болезнь; избирательность = здоровье.)

## 10. Верификация агентом

Модель принимается, если проходит тесты:

- ориентация сохранена (причинность цела);
- этажи коммутируют (укрупнение корректно);
- скелет = аттракторы своей же работы;
- симметрии устойчивы к ресемплингу (сигнал, а не артефакт);
- размеры частей подчиняются степенному закону (самоподобие);
- петля производства замкнута (живое, а не склад форм).

## 11. Статус утверждений

**Твёрдо:** редукция динамики в статику (строб); три оси декора; разведение ДНК/РНК; экономия за счёт малого алфавита; TDA / персистентная гомология как инструмент поиска ядер (первый названный в проекте инструмент оказался инструментом последнего слоя — круг сошёлся).

**Ставка (сознательная):** асимметричных пространственных ядер — конечное малое число, индексированное размерностью 2, 3, 4. Прецедент есть — конечные классификации неразложимых в геометрии (правильные многогранники: 3D → 5, 4D → 6). Но для *асимметричного* случая готовой классификации нет; это гипотеза автора.

**Первый конкретный шаг** (философия упёрлась в счёт): перечислить асимметричные неразложимые фигуры в 2D / 3D / 4D — гнать редукцию на многих объектах, смотреть, кластеризуются ли остатки в короткий список, индексированный размерностью. 4D-ядра — единственное место, где интуиция откажет, а гомология нет.

**Открытый параметр:** частота строба. Неверная частота → алиасинг (ложное движение из правильных кадров), РНК-последовательность станет артефактом сэмплирования. Та же монета, что тест устойчивости к ресемплингу.

## Глоссарий

- **Оцифровка** — извлечение скрытого графа ветвления и его динамики.
- **graph-unit / ядро** — неразложимая асимметричная пространственная фигура; базовая единица (аналог основания ДНК).
- **Цепочка** — 1D-последовательность ядер; код (генотип).
- **Ветвление** — свёрнутый из цепочки фенотип.
- **ДНК / РНК** — структура (фигура) / работа (стробо-последовательность фигур).
- **Строб** — стробоскопическое сэмплирование времени в статические кадры.
- **Три оси декора** — направления (причинность), параллели (кратность), строб (время).
- **Клетка** — операционально замкнутая единица (петля ДНК → работа → ДНК).
- **Слияние 1+1=1** — рекомбинация цепочек в новую цельность.
